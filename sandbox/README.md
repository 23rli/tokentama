# EcoPrompt Guardian — test sandbox

This folder is opened automatically in the **Extension Development Host** when you
press **F5**. It exists so the dev-host window has its **own** workspace (distinct
from the repo you're developing in), which lets EcoPrompt Guardian scope passive
capture to **this** window only.

## Try it

1. Open **Copilot Chat** in this window and send a normal prompt (no `@ecoprompt`).
2. Copilot answers as usual; within a few seconds the Guardian panel grades it.
3. Or click **⟳ Scan recent Copilot prompts** in the Guardian panel to grade your
   latest prompts immediately.

Watch **View → Output → "EcoPrompt Guardians"** to see capture happen live.
